#Android Studio 快捷键

**最近抽空整理了androidstudio 的快捷键，按照默认的keymap做了下面的表格，加粗的内容是常用的一些快捷键，有些理解不对的地方请指出来，感谢。**

##常用快捷键

名称 | 快捷键
--------|----
全局快速查找 | Double Shift
查找并打开文件 | Ctrl + Shift + N
最近打开文件列表 | Ctrl + E
目录文件导航栏 | Alt + Home
